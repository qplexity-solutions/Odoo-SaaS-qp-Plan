Manage the employees' tasks systematically!!

Task management is considered to be very important among employees as it helps them to become more effective and efficient which has a direct effect on productivity. Considering this we have brought a module Project Task Start Stop with the help of Which a user can easily manage a timesheet on task.

It enables you to keep a track on the work done by the employees in their working hours. Also, employees can work only at one task at a time. It also helps to calculate the total time spent between the state of start and stop for any task.

Highlighted Features

Below is the highlighted features of Odoo Project Task Start/Stop
Manage The Working Tasks For Employees
Calculate The Time Spent On each Task
Restricts User To Run One Task At A Time
View the duration of each task separately

Detailed Feature List

Below is the detailed list of features of Odoo Project Task Start/Stop

Manage The Working Tasks For Employees Effectively

  • Odoo Project Task Start Stop allows you to conveniently manage the tasks.
  • It also enables you to add an entry in the timesheet automatically when the task begins.

Calculate The Time Spent On The Task In Odoo

  • The module automatically calculates the total time spent between the start and stop status.
  • This module limits assigning the task to another user when that task is in working condition.

Restricts User To Run One Task At A Time In Odoo

  • This prevents the user from running two tasks concurrently.
  • Only the user to whom the task is assigned can start/stop his task.
