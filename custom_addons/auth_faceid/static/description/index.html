What does this module solve?

Default behavior

By default, you can log into odoo only with a username and password.

This is not very convenient because it is necessary to take care of the storage and relevance of passwords, as well as just remember them.

Behavior after install

After installing the module, it will be possible to log in to odoo using the facial recognition mechanism. The faces must first be uploaded to settings-users.

2-3 persons are usually enough for one user. If an employee wears glasses, preferably a photo with glasses and without. Protection against photo forgery is also implemented.

And you can also hide the standard password login interface or leave both options.

In addition, a logging mechanism is implemented that saves photos, time and user for manual control. The logging mechanism can also be turned on or off.
